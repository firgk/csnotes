## nrm
npm的镜像源管理工具，有时候国外资源太慢，使用这个就可以快速地在 npm 源间切换
npm install -g nrm
nrm ls
nrm use <registry>
nrm add <registry> <url>
nrm del <registry>



## nvm 
Node Version Manager（NVM） 是一种用于管理多个node.js版本的工具。
nvm -v
nvm list
nvm list available # 获取可以安装的版本
nvm install [版本号] # 安装某个版本
nvm uninstall [版本号] # 卸载某个版本
